# FaceRecognitionSystem
Steps to follow:

python version must be between 3.8-3.10

Make virtual environment:
    python -m venv venv

    venv\Scripts\activate.bat

import all the necessary libaries

    pip install cmake
    
    pip install dlib==19.24.2
    
    pip install face-recognition
    
    pip install mysql-connector-python
    
    pip install opencv-python

For Cmake install visual studio and install Cmake during installation process.

Include face_encoding file in main.py so that encoding is stored in db
